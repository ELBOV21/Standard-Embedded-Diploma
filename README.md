# Standard Embedded Systems Diploma

A collection of embedded systems projects completed as part of the Standard Embedded Systems Diploma program. Each project covers a different topic in embedded C development, circuit simulation, and microcontroller interfacing.

**Repository:** https://github.com/ELBOV21/Standard-Embedded-Diploma

---

## Projects Overview

### Project 1 - C Foundations
**Folder:** `Project_1/`

An introductory project focused on core C programming concepts applied to embedded systems. Includes a project specification PDF (`EDGES_C_Project_Standard_Final.pdf`) and a complete source archive.

**Contents:**
- `Diploma_Project_1.zip` - Source code archive
- `EDGES_C_Project_Standard_Final.pdf` - Project specification and requirements

---

### Project 2 - Microcontroller Interfacing
**Folder:** `Project_2/`

A mini project involving microcontroller peripheral interfacing, simulated using Proteus. Comes with a demo video, full source code, and the Proteus circuit file.

**Contents:**
- `Project_code/` - Eclipse workspace with C source files
- `Proteus_file/` - Proteus circuit simulation
- `Demo_Video/` - Video demonstration of the running project
- `Mini Project2.pdf` - Project brief and requirements
- `project_2_final.zip` - Full project archive

---

### Project 3 - Embedded Peripherals
**Folder:** `project_3/`

A more advanced mini project building on peripheral control and embedded C patterns. Includes two demo videos and both Eclipse and Proteus workspaces.

**Contents:**
- `Project_3_eclipse_WS/` - Eclipse workspace with C source files
- `Proteus_WS/` - Proteus circuit simulation
- `Project_3_final.mp4` - Final demo video
- `project_3_Demo.mp4` - Additional demo recording
- `Mini Project3.pdf` - Project brief and requirements
- `project_3.zip` - Full project archive

---

### Project 4 - Ultrasonic Distance Sensor (HC-SR04)
**Folder:** `project_4/`

An interfacing project using the HC-SR04 ultrasonic sensor to measure distance. Covers timer/counter configuration, echo pulse measurement, and display output on a microcontroller.

**Contents:**
- `project_4_WS/` - Eclipse workspace with C source files
- `proteus_WS/` - Proteus circuit simulation
- `HC-SR04 Ultrasonic MT Student Tutorial.pdf` - Sensor datasheet and tutorial
- `Mini_Project4.pdf` - Project brief and requirements
- `Project_4_final.mp4` / `Project_4_vid.mp4` - Demo videos
- `project_4.zip` - Full project archive

---

## Tools and Environment

| Tool | Purpose |
|------|---------|
| Eclipse IDE | C source code development and compilation |
| Proteus | Circuit schematic and simulation |
| AVR-GCC / AVR Toolchain | Compiler for ATmega microcontrollers |

---

## Getting Started

1. **Clone the repo**
   ```
   git clone https://github.com/ELBOV21/Standard-Embedded-Diploma.git
   ```

2. **Open a project in Eclipse** - import the relevant `*_WS` folder as an existing Eclipse workspace.

3. **Open the simulation** - load the Proteus file from the corresponding `Proteus_file/` or `proteus_WS/` folder to run the circuit simulation.

4. **Build and flash** - compile the project in Eclipse, then either run in Proteus or flash to a real ATmega board.
